# 🚖 TaxiEnv Pro — OpenEnv Hackathon Project

Welcome to **TaxiEnv Pro**, a custom Reinforcement Learning environment with a professional Gradio UI, built for the **Scaler x Hugging Face OpenEnv Hackathon**.

This project extends the classical taxi-task concept with:
- Clean, modern UI  
- Adjustable grid-size  
- Multiple episodes  
- Live reward charts  
- JSON state outputs  
- Built-in fallback environment  
- HF Spaces–ready design

## ✨ Features

### 🧠 Reinforcement Learning Environment
- Custom `TaxiEnv` with grid-world logic  
- Seed-based reproducibility  
- Step-by-step simulation  
- Fallback mock env if missing

### 🎨 Modern UI (Gradio)
- Clean professional theme  
- City-style background  
- Yellow taxi color palette  
- Responsive layout  

### 📊 Analytics
- Live reward line chart  
- Episode-wise reward breakdown  
- Success score estimation  
- Final frame / observation output
