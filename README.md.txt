# TaxiEnv - OpenEnv Hackathon Submission 🚀

## Tasks
1. Navigate 5x5 grid to pickup passenger
2. Deliver to destination (-5 step penalty, +20 success)

```bash
pip install -r requirements.txt

```

```bash
docker build -t taxi-env .
docker run -p 7860:7860 taxi-env
```
